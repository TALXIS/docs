# Signi (iSmlouva)

Signi je cloudová služba, určená primárně pro mobilní zařízení, která umožňuje vytváření, správu a uzavírání dokumentů ve 100% digitální podobě. Díky naší integraci ji můžete pohodlně využívat z Microsoft Power Automate (opens new window) a automatizovat tak podepisování dokumentů přímo z Power Platformy, SharePointu, Dynamics 365 nebo dalších systémů.

# Hlavní výhody

  • Podpora více workspaců (různých účtů např. pro Česko, Polsko, atd.)
  • Uživatel se přihlašuje svým pracovním účtem
  • Možnost řízení oprávnění přístupu k jednotlivým workspacům

# Začínáme

  1. Vygenerujte si API klíč pro váš workspace podle tohoto návodu (opens new window)
  2. Vyplňte následující formulář (opens new window), abychom pro Vás mohli vše nachystat
  3. Přidejte do svého prostředí oficiální konektor Signi (opens new window)
    • V případě potřeby se na nás můžete obrátit a my vám pomůžeme řešení nasadit.
